Zur Ausführung im Terminal, muss der Classpath angegeben werden, damit die Klasse "Settings" mit ausgeführt werden kann. 

Der Befehl könnte so aussehen (Ausführungszeichen und single quotes weglassen): 
"java -classpath 'der Pfad, in dem die sich die Settings Klasse/File befindet' 'pfad\Zombiegame.java"

zum Beispiel: java -classpath P:\Java\Projekte\OOP-JAVA-Lab\out\production\OOP-JAVA-Lab labor03.ZombieGame
